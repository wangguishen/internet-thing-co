# internet-thing-co

> A Vue.js project  -->  This is an enterprise management platform developed based on the background management system of the iot framework. It conducts order review for the application, terminal equipment, algorithm installation package, key and other series of operations of the enterprise, and forms a circular system with the background framework management system

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
